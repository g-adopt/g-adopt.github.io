Blankenbach 2D square convection - isoviscous

A simple 2-D square convection problem, from Blankenbach et al (1989), for execution in Firedrake, is displayed in Listing 1. The problem is incompressible (Bousinessq approximation), is heated from below and cooled from above, with closed, free-slip boundaries. Solutions are obtained by solving the Stokes equations for velocity and pressure, coupled to the energy equation for temperature and a suitable equation of state.
We have set up the problem using a bilinear (quadrilateral, Q2-Q1) elements for velocity and pressure, and Q2 elements for temperature, on a unit square mesh. In Listing 1, line 3 defines the mesh, with 40 quadrilateral elements in x and y directions. Important constants (Rayleigh Number, \(Ra\); Viscosity, \(\mu\); Thermal diffusivity, \(\kappa\)), in addition to the timestep and unit vector, are defined on lines 5-9. Lines 11-13 define the finite element function spaces and degree, using continuous Lagrange elements, on the given mesh. Test (lines 15-17) and trial (lines 19-26) functions are subsequently defined and initialised, with the UFL code for the Stokes and energy equations defined on lines 28-33. The resemblance to the mathematical formulation is immediately apparent. Dirichlet boundary conditions for velocity (boundaries identified by markers 1-4) and temperature (set to 1 at the base and 0 at the top, identified by markers 3 and 4, respectively) are specified on lines 36-37. We note that pressure is only defined up to a constant and, hence, we define the associated nullspace on line 38. The time-loop is initiated on line 43, with the Stokes system solved for velocity and pressure (z), using default solver options, and the energy equation solved using the same default options. Output, in VTU format is written on line 47, to a file initialised on line 40.

Busse 3D convection - isoviscous

Here we highlight the ease at which things can be tried in different geometries. We first examine and validate our setup in a 3-D Cartesian domain, for a steady state case with both constant viscosity -- case 1a from Busse el al. (1994). The domain is a box of dimensions \(1.0079 \times 0.6283 \times 1\). Boundary conditions for temperature are T = 0 at the surface (z = 1) and T = 1 at the base (z = 0), with insulating (homogeneous Neumann) sidewalls. No‐slip velocity boundary conditions are specified at the top surface and base of the domain, with free‐slip and no‐normal flow boundary conditions on all sidewalls. Initial conditions are chosen to produce a single ascending and descending flow, at \(x = y = 0\) and \((x = a)\), \((y = b)\), respectively. The Rayleigh number \(= 3 \times 10^{4}\).
